# 服装店记账系统

> 毕业设计作品 · 为妈妈而做的专属记账工具 · 纯离线 · 绿色主题

## 📖 项目简介

一款专为个体服装商户设计的安卓进销存管理应用。支持商品管理、销售记账、进货入库、账单查询、退货退款、库存查看、热销排行、利润趋势等功能。**完全离线运行**，所有数据存储在本地数据库，无需网络。

**真实用户驱动**：本项目以我的妈妈（服装摊主）为真实用户，从需求分析、交互设计到功能实现，全程基于她的实际经营场景进行定制开发。

### 🎯 用户画像

- **种子用户**：个体服装摊主（本项目为我的妈妈）
- **使用场景**：服装市场摊位，环境嘈杂，网络不稳定
- **核心需求**：快速记账、库存管理、经营分析、操作极简

## ✨ 核心功能

| 模块 | 功能 |
| :--- | :--- |
| 🏠 **首页** | 店铺名称动态显示、快捷指令记账、分类列表（动态加载）、底部导航 |
| 📦 **商品管理** | 添加商品（拍照+动态属性标签）、商品列表、点击商品弹出销售对话框 |
| 🏷️ **属性管理** | 属性组增删（风格/面料/颜色/尺码等）、属性值批量添加/单独删除、智能提示、二次确认 |
| 📂 **分类管理** | 动态增删分类、预置6个默认分类、二次确认删除 |
| 💰 **销售记账** | 点击记账（支持临时改价）、快捷指令记账（输入“卖了连衣裙一件90块”自动解析并记账） |
| 📥 **进货入库** | 选择商品后自动展示属性组合、逐行填写数量和进价、实时计算总进货金额、二次确认入库 |
| 📊 **账单查询** | 日/月/年维度切换、日历选择器（日模式显示完整日历，月模式12个月，年模式年份范围）、默认隐藏金额（查询后才显示）、点击销售记录查看详情 |
| ↩️ **退货退款** | 销售流水左滑弹出退款确认、二次确认后退款、销售额和毛利自动更新 |
| 📦 **库存查看** | 实时计算库存（进货－销售＋退货）、低库存预警（≤5件红色标识） |
| 🏆 **热销排行** | 按销量降序排列、显示销售额和毛利、前三名排名数字高亮 |
| 📈 **利润趋势** | 按月汇总销售额、毛利、笔数 |
| 💾 **数据管理** | 导出数据库文件到下载文件夹、从备份文件恢复（需授权码 `0831`）、每日自动备份到本地并保留7天、一键初始化系统恢复出厂状态 |
| 🔊 **音效反馈** | 卖出、入库、退款、删除等操作有差异化提示音、全局按钮点击音效、声音开关可控 |
| 🔒 **安全机制** | 所有删除和退款操作需二次确认、账单数据默认隐藏保护隐私、授权码保护敏感操作 |
| 🎨 **视觉风格** | 绿色渐变背景、28dp大圆角按钮、胶囊形底部导航、半透明白色卡片、统一沉浸式状态栏 |

### 数据库设计

| 表名 | 说明 |
| :--- | :--- |
| `products` | 商品基础信息（含图片路径、进价、售价） |
| `categories` | 商品分类（支持二级分类） |
| `attribute_groups` | 属性组（风格/面料/颜色/尺码/季节等，用户可自定义） |
| `attribute_values` | 属性值（淑女风/纯棉/红色/M码等） |
| `product_attributes` | 商品与属性的多对多关联 |
| `sales` | 销售记录（含退款标记、进价、售价） |
| `purchases` | 进货记录（含属性描述） |

> 库存通过 `累计进货 - 累计销售（不含退款）` 实时计算，保证数据一致性。

## 🚀 使用说明

### 安装方式
1. 用 Android Studio 打开项目
2. Build → Build Bundle(s) / APK(s) → Build APK(s)
3. 将 `app/build/outputs/apk/debug/app-debug.apk` 安装到安卓平板或手机
4. 首次启动自动初始化预置数据（默认分类、属性、示例商品）

### 基本操作流程
- **第一次使用**：去“我的”→ 添加商品（定义名称、分类、属性标签、售价）
- **进货入库**：去“进货”→ 选择商品 → 填写尺码/颜色/数量/进价 → 确认入库
- **日常销售**：首页点击分类 → 点击商品 → 确认数量/售价 → 确认卖出
- **看账**：去“账单”→ 选择日期 → 查询 → 查看销售额、毛利、流水
- **退货**：账单页面左滑销售记录 → 确认退款
- **库存提醒**：去“我的”→ 库存查看 → 低库存商品红色预警

### 快捷指令
首页输入框支持文字指令记账（无需点击分类和商品）：
- `卖了连衣裙一件90块` → 自动解析并弹出确认框，确认后记账
- `进了连衣裙5件60块` → 进货入库
- `退了连衣裙一件` → 退货退款

### 数据管理
- **自动备份**：每天首次启动时自动备份到 `Download/记账备份/`，保留最近7天
- **手动导出**：“我的”→ 导出备份 → 文件保存在 `Download/` 文件夹
- **恢复数据**：“我的”→ 恢复备份 → 选择备份文件 → 输入授权码 `0831` → 确认恢复
- **初始化系统**：“我的”→ 初始化系统 → 输入授权码 `0831` → 确认 → 重启后恢复出厂状态

## ⚠️ 注意事项

- 本应用为纯离线设计，所有数据存储在设备本地
- 建议定期导出备份，通过微信等渠道发送给自己保存
- 删除操作均有二次确认，防止误触
- 商品进价录入后不可修改（保护历史数据正确性）
- 账单数据默认隐藏，需主动查询才显示（保护隐私）
- 恢复备份和初始化系统需要授权码 `0831`
