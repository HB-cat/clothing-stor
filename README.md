# 服装店记账系统

> 毕业设计作品 · 为妈妈而做的专属记账工具 · 纯离线 · 绿色主题

## 📖 项目简介

一款专为个体服装商户设计的安卓进销存管理应用。支持商品管理、销售记账、进货入库、账单查询、退货退款、库存查看、热销排行、利润趋势等功能。**完全离线运行**，所有数据存储在本地数据库，无需网络。

### 🎯 用户画像

- **种子用户**：个体服装摊主（本项目为我的妈妈）
- **使用场景**：服装市场摊位，环境嘈杂，网络不稳定
- **核心需求**：快速记账、库存管理、经营分析

## ✨ 核心功能

| 模块 | 功能 |
| :--- | :--- |
| 🏠 **首页** | 店铺名称、快捷指令、分类列表、底部导航 |
| 📦 **商品管理** | 添加商品（拍照+动态属性）、商品列表、点击销售 |
| 🏷️ **属性管理** | 属性组增删、属性值批量添加/删除、二次确认 |

| 💰 **销售记账** | 点击记账（支持改价）、快捷指令记账（规则解析） |

| 📊 **账单查询** | 日/月/年维度切换、日历选择器、默认隐藏数据、左滑退款 |

| 📈 **利润趋势** | 按月汇总销售额、毛利、笔数 |
| 💾 **数据管理** | 导出/恢复备份（授权码保护）、每日自动备份（保留7天）、一键初始化 |
| 🔊 **音效反馈** | 操作提示音、全局点击音效、可开关 |
| 🎨 **视觉风格** | 绿色渐变背景、大圆角按钮、胶囊导航、半透明白色卡片 |

### 数据库设计

| 表名 | 说明 |
| :--- | :--- |
| `products` | 商品基础信息（含图片路径） |
| `categories` | 商品分类 |
| `attribute_groups` | 属性组（风格/面料/颜色/尺码等） |
| `attribute_values` | 属性值 |
| `product_attributes` | 商品与属性的多对多关联 |
| `sales` | 销售记录（含退款标记） |
| `purchases` | 进货记录（含属性描述） |

> 库存通过 `累计进货 - 累计销售（不含退款）` 实时计算，保证数据一致性。

## 🚀 使用说明

### 安装方式
1. 用 Android Studio 打开项目
2. Build → Build APK
3. 将 `app-debug.apk` 安装到安卓平板或手机
4. 首次启动自动初始化预置数据

### 基本操作流程
- **第一次使用**：去"我的"添加商品（定义名称、属性、售价）
- **进货入库**：去"进货"选择商品，填写数量进价
- **日常销售**：首页点分类→点商品→确认卖出
- **看账**：去"账单"选择日期查询
- **退货**：账单页面左滑销售记录

### 快捷指令
首页输入框支持文字指令记账：
- `卖了连衣裙一件90块` → 卖出1件连衣裙，售价90
- `进了连衣裙5件60块` → 进货5件连衣裙，进价60
- `退了连衣裙一件` → 退回最近一笔该商品销售

### 数据导出
- **自动备份**：每天首次启动时自动备份到 `Download/记账备份/`，保留最近7天
- **手动导出**："我的" → 导出备份 → 文件保存在下载文件夹
- **恢复数据**："我的" → 恢复备份 → 需输入授权码 `0831`

## ⚠️ 注意事项

- 本应用为纯离线设计，所有数据存储在设备本地
- 建议定期导出备份，通过微信等渠道发送给自己保存
- 删除操作均有二次确认，防止误触
- 进价录入后不可修改（保护历史数据正确性）
- 账单数据默认隐藏，需主动查询才显示（保护隐私）
