# 服装店记账系统

> 毕业设计作品 · 为妈妈而做的专属记账工具

## 📖 项目简介

一款专为个体服装商户设计的安卓进销存管理应用。支持商品管理、销售记账、进货入库、账单查询、退货退款、库存查看、热销排行、利润趋势等功能。**纯离线运行**，所有数据存储在本地数据库，无需网络。

### 🎯 用户画像

- **种子用户**：个体服装摊主（本项目为我的妈妈）
- **使用场景**：服装市场摊位，环境嘈杂，网络不稳定
- **核心需求**：快速记账、库存管理、经营分析

## ✨ 核心功能

| 模块 | 功能 |
| :--- | :--- |
| 🏠 **首页** | 动态标题、分类网格（动态加载）、快捷指令记账 |
| 📦 **商品管理** | 添加商品（动态属性标签）、商品列表、点击销售 |
| 🏷️ **属性管理** | 属性组增删、属性值批量添加/删除、智能提示、二次确认 |
| 📂 **分类管理** | 动态增删分类、预置6个默认分类 |
| 💰 **销售记账** | 点击记账（支持改价）、快捷指令记账（规则解析） |
| 📥 **进货入库** | 多属性行批量入库、自动计算总价 |
| 📊 **账单查询** | 日/月/年维度切换、日历选择器、默认隐藏数据 |
| ↩️ **退货退款** | 左滑退款、二次确认、销售额自动更新 |
| 📦 **库存查看** | 实时计算库存、低库存预警（≤5件） |
| 🏆 **热销排行** | 按销量降序、显示销售额和毛利 |
| 📈 **利润趋势** | 按月汇总销售额、毛利、笔数 |
| 💾 **数据管理** | 导出/恢复备份（授权码保护）、每日自动备份（保留7天） |
| ⚙️ **个性化** | 店铺名称修改、菜单拖拽排序 |

## 🛠️ 技术栈

| 层级 | 技术 |
| :--- | :--- |
| **开发语言** | Java |
| **UI框架** | Android XML Layout |
| **数据库** | Room（SQLite） |
| **最低SDK** | Android 11（API 30） |
| **测试环境** | Pixel 2 API 33 模拟器 / 酷派平板 Android 15 |

### 数据库设计

| 表名 | 说明 |
| :--- | :--- |
| `products` | 商品基础信息 |
| `categories` | 商品分类（支持二级分类） |
| `attribute_groups` | 属性组（风格/面料/颜色/尺码等） |
| `attribute_values` | 属性值（淑女风/纯棉/红色/M码等） |
| `product_attributes` | 商品与属性的多对多关联 |
| `sales` | 销售记录（含退款标记） |
| `purchases` | 进货记录（含属性描述） |

> 库存通过 `累计进货 - 累计销售（不含退款）` 实时计算，保证数据一致性。
