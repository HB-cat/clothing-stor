# 服装店记账系统

> 毕业设计作品 · 为妈妈而做的专属记账工具 · 纯离线 · 绿色主题

## 📖 项目简介

一款专为个体服装商户设计的安卓进销存管理应用。支持商品管理、销售记账、进货入库、账单查询、退货退款、库存查看、热销排行、利润趋势等功能。**完全离线运行**，所有数据存储在本地数据库，无需网络。

### 🎯 用户画像

- **种子用户**：个体服装摊主（本项目为我的妈妈）
- **使用场景**：服装市场摊位，环境嘈杂，网络不稳定
- **核心需求**：快速记账、库存管理、经营分析

## ✨ 核心功能

| 模块 | 功能 |
| :--- | :--- |
| 🏠 **首页** | 店铺名称、快捷指令、分类列表、底部导航 |
| 📦 **商品管理** | 添加商品（拍照+动态属性）、商品列表、点击销售 |
| 🏷️ **属性管理** | 属性组增删、属性值批量添加/删除、二次确认 |
| 📂 **分类管理** | 动态增删分类、预置默认分类 |
| 💰 **销售记账** | 点击记账（支持改价）、快捷指令记账（规则解析） |
| 📥 **进货入库** | 属性组合批量入库、自动计算总价 |
| 📊 **账单查询** | 日/月/年维度切换、日历选择器、默认隐藏数据、左滑退款 |
| 📦 **库存查看** | 实时计算库存、低库存预警（≤5件） |
| 🏆 **热销排行** | 按销量降序，前三名高亮 |
| 📈 **利润趋势** | 按月汇总销售额、毛利、笔数 |
| 💾 **数据管理** | 导出/恢复备份（授权码保护）、每日自动备份（保留7天）、一键初始化 |
| 🔊 **音效反馈** | 操作提示音、全局点击音效、可开关 |
| 🎨 **视觉风格** | 绿色渐变背景、大圆角按钮、胶囊导航、半透明白色卡片 |

## 🛠️ 技术栈

| 层级 | 技术 |
| :--- | :--- |
| **开发语言** | Java |
| **UI框架** | Android XML Layout + 自定义样式 |
| **数据库** | Room（SQLite） |
| **最低SDK** | Android 11（API 30） |
| **测试环境** | Pixel 2 API 33 模拟器 / 酷派平板 Android 15 |

### 数据库设计

| 表名 | 说明 |
| :--- | :--- |
| `products` | 商品基础信息（含图片路径） |
| `categories` | 商品分类 |
| `attribute_groups` | 属性组（风格/面料/颜色/尺码等） |
| `attribute_values` | 属性值 |
| `product_attributes` | 商品与属性的多对多关联 |
| `sales` | 销售记录（含退款标记） |
| `purchases` | 进货记录（含属性描述） |

> 库存通过 `累计进货 - 累计销售（不含退款）` 实时计算，保证数据一致性。
