## 一、产品概述

### 1.1 产品定位

一款专为个体服装商户设计的安卓离线进销存管理应用。支持商品管理、销售记账、进货入库、账单查询、退货退款、库存查看、经营分析等功能。v1.4当前版本所有数据存储在设备本地SQLite，v1.5规划迁移至云端MySQL以支持多用户协同。

### 1.2 目标用户

**种子用户**：我妈，48岁，服装摊主，非技术背景。

**经营环境**：服装市场摊位，环境嘈杂，网络不稳定。

**核心痛点**：
- 手工纸笔记账易丢失
- 库存靠翻货架，不知道什么款好卖
- 晚上收摊后需花大量时间对账
- 缺乏经营数据分析能力，不知道什么该补货、什么该清仓

### 1.3 核心价值

- **10秒完成记账**：点击商品 → 确认售价 → 完成销售
- **离线可用**：v1.4纯本地运行，无需网络
- **灵活扩展**：分类、属性完全由用户自定义
- **隐私保护**：账单数据默认隐藏，查询后才显示
- **经营洞察**：规则引擎自动分析数据，生成经营建议

## 四、交互设计

### 4.1 核心设计原则

- **拇指友好**：所有可点击元素高度至少48dp，主按钮56dp
- **三步完成**：核心记账操作不超过三步
- **点选代替输入**：分类、属性通过下拉框或标签选择
- **出错可逆**：所有删除、退款等敏感操作均需二次确认
- **默认不暴露数据**：账单需主动查询才显示金额
- **结论先行**：经营分析用大号文字给出精简结论，想看详细数据再点击进入

### 4.2 视觉风格

- **主题色**：稳重绿（#43A047）
- **状态栏**：沉浸式绿色（与标题栏融为一体）
- **页面背景**：绿色渐变（#C8E6C9 → #A5D6A7 → #F1F8E9）
- **标题栏**：深绿色背景+白色文字（56dp高）
- **大按钮**：28dp大圆角、绿色实心白字（56dp高）
- **中等按钮**：28dp圆角、绿色实心白字（44dp高）
- **卡片**：16dp圆角、半透明白色背景、轻阴影
- **底部导航**：白色胶囊容器、选中绿底白字、未选中透明底灰字
- **对话框按钮**：绿色实心白字、圆角

### 4.3 导航体验

- **底部导航固定**：所有主页面底部导航栏固定在屏幕底部，不随内容滚动
- **统一切换动画**：点击导航按钮时使用淡入淡出动画
- **当前页面标识**：选中的导航按钮绿色实心+白字，其他透明底+灰字

### 4.4 经营分析交互（v1.4新增）

- **主页分层**：今日快报→趋势图→各模块精简结论+按钮
- **单入口设计**：每个模块一个"查看"按钮，靠右对齐，半透明圆角样式
- **图表横屏**：趋势/热销/时段点击进入横屏全屏图表，含数值标注
- **列表弹窗**：滞销/补货点击弹出对话框列表

## 六、数据库设计

| 表名 | 说明 |
| :--- | :--- |
| `products` | 商品基础信息（含图片路径、进价、售价） |
| `categories` | 商品分类 |
| `attribute_groups` | 属性组（风格/面料/颜色等） |
